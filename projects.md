# Projects

Makerspace improvement projects — organized efforts to upgrade facilities, relocate, or improve member experience.

## Active Projects

- [2026 Relocation](/projects/2026-relocation) — Facility move project (details private to members)

## Project Page Conventions

Each project uses a sibling landing page and a folder for supporting documents:

```
projects/
  <project-name>.md          ← public landing page / summary
  <project-name>/            ↔ detailed documents, meeting notes, files
```

Sensitive project documents are kept private via Wiki.js Guest group page rules.
