Preset Codes

Call a preset to pan and zoom to that remembered position. But there are number of somewhat secret preset codes:

 

Factory Reset

Avoid this unless there's problems, will lose all saved positions!

  • Call Preset 115
  • Set Preset 40
  • Call Preset 33 to just remove all Preset PTZ points

 

Cruise Mode - Travel from point to point continuously

  • Call Preset 41: "The 1 cruise line (preset positions 1-32)"
  • Call Preset 42: "The 2 cruise line (preset positions 17-32)"
  • Set Preset 93+ Set N (the value of N is 20-255): “Tracking mode: Tracking time setting”
  • Set Preset 94+ Set N (the value of N is 1-255): “Tracking mode: Return time setting”
  • Call Preset 93: “Tracking mode: Turn on cruise line tracking”

 

Infrared Light

  • Call 82 to turn on IR-CUT
  • Call 83 to turn off IR-CUT
  • Call 81 to set IR-CUT back to automatic

This can be useful if the camera is blurry or the picture is unusable at night. Call 82  (turn on the infrared light) and wait for about 5s, then call 83 (turn off the infrared light). Call 82 & 83 repeatedly for several times to see if the camera’s pic color can return to normal. Then, please remember to call 81 (automatic color).

 

Human Tracking

  • Call Preset 92. 
  • Only works when Intelligent Tracking AND Motion Detection is on. 
  • Only Tracks humans. 
  • This mode does not persist after reboot - need some way to automate it
  • If the camera ignores calling 92, seems to only be fixed by a reboot.
