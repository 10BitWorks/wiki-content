# Vendored Services

External cloud platforms, SaaS subscriptions, and APIs used to run 10BitWorks.

## Active Integrations

- **GitHub** — Code hosting, pull request workflows, and wiki repository hosting
- **Slack** — Main team communication and automated alert channels
- **Porkbun** — Domain registrar and DNS hosting for `10bit.works` and `10bitworks.org`
- **Google Workspace** — Domain email routing, Google Drive document storage, and calendar embeds
- **Tailscale (Commercial)** — Cloud-managed coordination coordination for remote network overlay
- **Coolify cloud hooks** — Deployment triggers and backup integrations
