# Self-Hosted Services

Applications running in Docker containers on local servers (Zelda, Link) or cloud hosts (Oracle Cloud).

## Active Services

- [AppFlowy](/app-services/self-hosted/appflowy) — Collaborative workspace and notes
- [Authentik](/app-services/self-hosted/authentik) — Identity provider (legacy)
- [BookStack](/app-services/self-hosted/bookstack) — Documentation wiki
- [Dozzle](/app-services/self-hosted/dozzle) — Docker log viewer
- [Frigate](/app-services/self-hosted/frigate) — Network video recorder for security cameras
- [Kanidm](/app-services/self-hosted/kanidm) — Identity provider & SSO
- [Loomio](/app-services/self-hosted/loomio) — Group decision-making
- [MCP](/app-services/self-hosted/mcp) — Model context protocol server
- [n8n](/app-services/self-hosted/n8n) — Workflow automation
- [Plane](/app-services/self-hosted/plane) — Project management
- [ResourceSpace](/app-services/self-hosted/resourcespace) — Digital asset management
- [Vaultwarden](/app-services/self-hosted/vaultwarden) — Password manager
- [Vikunja](/app-services/self-hosted/vikunja) — Task and project tracking
- [Wiki VM on Zelda](/app-services/self-hosted/wiki-vm-on-zelda) — Wiki.js instance
- [WordPress + CiviCRM](/app-services/self-hosted/wordpress-civicrm) — Membership database and public site
- [Xibo](/app-services/self-hosted/xibo) — Digital signage management
- [Zammad](/app-services/self-hosted/zammad) — Support ticketing
