# Ray

Ray built the original front door access control hardware and software for 10BitWorks, including both the frontdoor and cobalt Raspberry Pi controllers and the early card provisioning workflow.

## Roles & Contributions

- **Access Control Pioneer**: Designed and built the original RFID door controller systems.
- **Cardmaker Pi**: Contributed to the early badge provisioning workflow.
